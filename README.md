<h1 align="center">📊 Tea vs Coffee Global Data Analysis Dashboard</h1>

<p align="center">
An end-to-end data analysis project exploring global beverage consumption patterns using Excel and Power BI.
</p>

<hr>

<h2>🚀 Project Summary</h2>
<p>
Built an end-to-end <b>data analysis and visualization project</b> to explore global consumption patterns of 
<b>tea vs coffee</b> using a structured dataset with 30+ features.
</p>

<p>
The project focuses on transforming raw data into <b>actionable business insights</b> using 
<b>Microsoft Excel and Power BI</b>, highlighting how demographic, behavioral, and health factors influence beverage preferences.
</p>

<hr>

<h2>🎯 What I Did</h2>
<ul>
<li>Cleaned and preprocessed raw CSV data using <b>Microsoft Excel</b> (handled missing values, removed duplicates, standardized data)</li>
<li>Performed <b>Exploratory Data Analysis (EDA)</b> to identify trends and patterns</li>
<li>Built an <b>interactive Power BI dashboard</b> with filters and drill-down capabilities</li>
<li>Created <b>DAX measures</b> for KPIs such as average cups/day, spending, and satisfaction score</li>
<li>Conducted <b>comparative and correlation analysis</b> (Tea vs Coffee, health vs lifestyle, spending vs income)</li>
</ul>

<hr>

<h2>📊 Key Insights</h2>
<ul>
<li>Identified regional preferences for <b>tea vs coffee</b></li>
<li>Analyzed relationship between <b>stress levels and caffeine consumption</b></li>
<li>Discovered how <b>income level impacts beverage spending</b></li>
<li>Evaluated <b>customer loyalty vs experimentation behavior</b></li>
<li>Explored correlation between <b>sleep, health metrics, and drink choice</b></li>
</ul>
